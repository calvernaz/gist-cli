gist-cli
========

## Installation

 # Requesites
  
     * [Curl] (http://curl.haxx.se/)
     * [GitHub API token access] (http://developer.github.com/v3/oauth/)

   $ npm install gist-cli


## Configuration

 # Setup

   * Edit config.json file and insert your developer key

   $ {
	   "Token": "INSERT_YOUR_KEY_HERE" 
     }
